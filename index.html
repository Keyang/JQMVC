Pick Up Your MVC Gears in JS

JQMVC provides a universal way to write JS based application
JQMVC is a highly extendible MVC framework for Javascript based environment. The power of JQMVC is it keeps same code style for different Javascript environment such as Browsers, Node.js etc. The core of JQMVC is highly abstracted standard MVC pattern. Besides, it allows developers to add self-defined plugins to support more view typs and proxies types. Developers could easily write JS based application in same style for Web Application, Hybird Mobile Application, WebOs Application, Node.Js Application etc.
