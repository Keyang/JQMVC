Pick Up Your MVC in JS

JQMVC provides a universal way to write JS based application
JQMVC extends MVC to different areas of Javascript applications using a series of universal coding methods. With different combinations of Model-View-Controller instances, developers could easily write JS based application in same paradigm for Mobile Website, Hybird Mobile Application, Website, Node.Js etc.
